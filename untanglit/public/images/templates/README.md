# Template preview images

Each template folder can have a **homepage preview** used on the `/templates` listing:

- `construction/homepage-preview.webp` – screenshot of the Construction template homepage
- `indie-author/homepage-preview.webp` – screenshot of the Indie Author template homepage
- `restaurant/homepage-preview.webp` – screenshot of the Restaurant template homepage
- `auto-shop/homepage-preview.webp` – screenshot of the Auto Shop template homepage

**How to add/update:** Open the template URL in your browser (e.g. `/templates/construction`), take a viewport screenshot (e.g. DevTools → right‑click page → “Capture screenshot”, or your OS shortcut), and save it as `homepage-preview.webp` (or `.png`) in the matching folder above. A 1200×800 or similar viewport works well.
